# HANGMAN GAME

--------------------

## How to install

1- Download the zip file or clone the repository 

2- Navigate to the Hangman directory, open a terminal and run these commands:

    mvn clean

Then: 

    mvn compile

Finally: 

    mvn exec:java

Else you can run them in the same command line:

    mvn clean compile exec:java
